Prework Study Guide

HTML

  • HTML is composed of elements, which structure the webpage and define content
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS helps Web developers create a uniform look across several pages of a Web site.
  • CSS makes it easy to change styles across several pages at once.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • GIT STATUS: checks status
  • GIT CHECKOUT (plus name of branch): changes branch
  • GIT BRANCH: list branches (also can change branch with name attached)
  • GIT PULL ORIGIN (plus branch name): receives new information from a branch into local environment.

JAVASCRIPT

  • Javascript is a scripting languages, primarily used on the Web.
  • It is used to enhance HTML pages and is commonly found embedded in HTML code.
